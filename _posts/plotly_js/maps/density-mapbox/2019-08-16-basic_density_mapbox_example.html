--- name: Stamen Terrain Tile language: plotly_js suite: tile-density-heatmaps order: 6 sitemap: false layout: base arrangement: horizontal markdown_content: | > Mapbox traces are deprecated and may be removed in a future version of Plotly.js. Earlier examples use traces that render with [Maplibre GL JS](https://maplibre.org/maplibre-gl-js/docs). These traces were introduced in Plotly.js 2.35.0 and replace Mapbox-based tile maps, which are now deprecated. Here's one of the earlier examples written using the Mapbox-based `densitymapbox` trace. --- var data = [{type: 'densitymapbox', lon: [10, 20, 30], lat: [15, 25, 35], z: [1, 3, 2]}]; var layout = {width: 600, height: 400, mapbox: {style: 'https://tiles.stadiamaps.com/styles/stamen_watercolor.json?api_key=YOUR-API-KEY'}}; Plotly.newPlot('myDiv', data, layout);
